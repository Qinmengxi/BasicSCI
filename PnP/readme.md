1、Clone repository  
```bash  
git clone https://github.com/Qinmengxi/BasicSCI.git  
```  

2、Install dependent packages
```bash  
pip install -r requirements.txt  
```  

3、Use GAP-TV model
```bash  
python gap_tv.py 
```  

4、Use PnP (FFDNet) model
```bash  
python gap_ffdnet.py
```  

5、Use PnP (FastDVDnet) model
```bash  
python gap_fastdvdnet.py
```
